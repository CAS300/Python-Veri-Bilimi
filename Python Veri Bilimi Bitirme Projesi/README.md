# Customer Churn Analizi (Bitirme Projesi)

Bu proje, müşteri kaybını (Churn) tahmin etmek için veri analizi (EDA) ve Lojistik Regresyon modeli oluşturma adımlarını içerir.

---

## Projenin Amacı
Amaç, bir telekom şirketinin müşteri verilerini analiz ederek hangi müşterilerin hizmetten ayrılma (Churn) eğiliminde olduğunu belirlemektir.  
Veri seti: `Customer-Churn.csv`

---

## Kullanılan Kütüphaneler
Projede yalnızca ders kapsamında işlenen temel kütüphaneler kullanılmıştır:

- pandas  
- numpy
- matplotlib
- scikit-learn

---

## Çalıştırma Adımları
1. Python ortamını aç.  
2. Gerekli kütüphaneleri yükle
3. Komutu çalıştır: python bitirme_projesi.py
